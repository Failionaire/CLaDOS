Overview

A REST API for managing tasks across teams. Users can create projects, invite colleagues, assign tasks, set due dates, and get notified of changes. Deployable as a standalone service behind an existing API gateway.

backend-only REST PostgreSQL

User Stories

  • As a user, I can register and log in to manage my tasks.
  • As a user, I can create, update, and delete tasks within a project.
  • As a team lead, I can assign tasks to other users and set due dates.
  • As a user, I can filter tasks by status, assignee, and due date.
  • As a user, I receive a notification when a task is assigned to me.

Must Not Do

  • Store or process payment information.
  • Expose any user data outside their organisation boundary.

Acceptance Criteria

  • All endpoints respond under 200 ms at p95.
  • All data is scoped to the authenticated user's organisation.
  • API is documented with OpenAPI 3.1.
